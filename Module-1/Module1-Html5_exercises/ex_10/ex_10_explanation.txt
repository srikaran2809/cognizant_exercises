Live Style Modification Using Inspect Element:

   * Open the webpage in Google Chrome.
   * Right-click on the element experiencing a layout issue and select Inspect.
   * In the Elements tab of Chrome DevTools, examine the HTML structure and corresponding CSS rules.
   * Modify CSS styles live to test different UI layouts.
   * After confirming changes improve the layout, replicate the adjustments in the source CSS files using VS Code.

View Logs via the Console Tab:

   * Navigate to the Console tab within Chrome DevTools.
   * Review any error messages, warnings, or custom log messages generated by the JavaScript <script> tags.
   * Utilize console.log(),console.error() and console.warn() to trace and understand execution flow and potential issues.
   * Fix the root causes of script errors by updating the JavaScript code in VS Code accordingly.
